# Book Recommender

A Streamlit app that recommends books using content-based and collaborative filtering. It analyzes your reading history, favorite authors, friendsâ€™ preferences, and similar users to suggest books. Powered by the BookCrossing dataset, with user login and friend management features.

## Features
- Personalized book recommendations
- Trending books among friends
- Suggestions from users with similar interests
- User login and friend management
- Visual book info with cover images

## Tech Stack
- Python
- Streamlit
- Pandas

## Data
- BookCrossing dataset (Books, Ratings, Users)

## Usage
1. Install requirements: `pip install streamlit pandas`
2. Run the app: `streamlit run app.py`
3. Use the sidebar to log in and manage friends
4. Explore personalized book recommendations
